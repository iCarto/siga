Różnica

Geoproces ten działa dla dwóch warstw: warstwa wejściowa oraz warstwa przysłaniająca. Geoproces ten nazywany jest również „przestrzenna różnica”, ponieważ zostawia geometrię obiektów nie pokrywających się w obiektach warstwy przysłaniającej.

Geoproces ten działa jedynie z warstwami obiektów powierzchniowych. Warstwa wynikowa będzie miała takie same atrybuty jak warstwa wejściowa.

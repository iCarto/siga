Differenz (Difference)

Die Funktion „Differenz“ verschneidet zwei Vektorthemen A („Eingabe-Ebene“) und B („Overlay-Ebene“) derart miteinander, so dass das Ergebnis nur noch diejenigen Flächen enthält, die lediglich in Ebene A, nicht aber in Ebene B enthalten sind. Die Funktion „Differenz“ wird daher auch „spatial NOT“ genannt und ist das Gegenteil der Funktion „Daten ausschneiden“. Als Eingabeparameter können ausschließlich Polygonthemen verwendet werden. Das Ergebnis übernimmt das Tabellenschema Eingabe-Ebene A.

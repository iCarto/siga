Разность (Difference)

Этот инструмент работает с двумя слоями - исходным слоем и операционным слоем. Операция соответствует запросу "Spatial NOT". Из объектов вырезаются части находящиеся в границах объектов операционного слоя.

Исходный и операционный слой должны быть полигональными. Выходной слой будет иметь идентичную схему исходному.

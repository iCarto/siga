Desberdintasuna

Desberdintasun geoprozesuak bi geruzekin egiten du lan: sarrera geruza eta gainjartze geruza.

‘NOT’ espaziala bezala ezagutzen da: Sarrera geruzan dauden eta gainjartze geruzan EZ dauden guneak aukeratzen ditu.

Bai sarrera geruza eta baita gainjartze geruza ere poligonodunak izan behar dira. Irteera geruzak sarrerakoaren taula berdina izango du; desagertzen diren poligonoen datuak ez dira agertuko.

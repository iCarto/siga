Rozdíl

Tato prostorová překryvná operace pracuje se dvěmi vrstvami: vstupní a překryvná vrstva. Obvykle se to nazývá “Prostorové NOT”, protože ponechává nepřekrývající se geoprvky.

Pracuje pouze s polygony (vstup a překryvná vrstva). Výsledná vrstva bude mít zachovánu stejnou strukturu atributů jako vstup.

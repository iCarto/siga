Zusammenführen (Dissolve)

Diese Funktion führt diejenigen Flächen einer Eingabe-Ebene vom Typ Polygon zusammen, die über den selben Wert für ein bestimmtest Attribut verfügen. Der Benutzer kann optional festlegen, dass ausschließlich direkt aneinander grenzende Polygone zusammengeführt werden. Zur Zusammenfassung der Attribute stehen die Funktionen Minimum, Maximum, Mittelwert sowie Summe zur Verfügung.

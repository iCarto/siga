Daten ausschneiden (Clip)

Diese Funktion benötigt als Eingabeparameter zwei Vektorebenen A („Eingabe-Ebene“) und B („Clip-Ebene“). Beide Ebenen werden durch die Funktion „Daten ausschneiden“ so miteinander verschnitten, dass das Ergebnis nur diejenigen Flächen enthält, die innerhalb der Clip-Ebene liegen. Das Tabellenschema der Eingabe-Ebene A bleibt in der neuen Vektorebene erhalten. Die Funktion „Daten ausschneiden“ kann beispielsweise eingesetzt werden, um den Umgriff einer Ebene auf einen bestimmten Untersuchungsraum zu begrenzen,

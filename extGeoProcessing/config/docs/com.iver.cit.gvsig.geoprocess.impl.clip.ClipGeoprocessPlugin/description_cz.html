Ořez

Tato prostorová překryvná operace ořezává prostorovou část vstupní vrstvy podle hranic geoprvků ořezávající vrstvy. Výsledná vrstva je tvořena prvky, jejichž geometrie se nachází uvnitř hranic ořezávajících geoprvků.

Výstupní vrstva zachovává atributy vstupní vrstvy.

    To může být užitečné při vymezení geografické oblasti zájmu (např. ořezání národních dat na rozsah vybraného okresu).

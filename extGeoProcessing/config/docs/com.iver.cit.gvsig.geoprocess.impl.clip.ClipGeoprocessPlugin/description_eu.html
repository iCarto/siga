Ebaki

Geoprozesu honek, geruza bektorial bat (puntu, lerro edo poligonoak) ebakitzeko balio du.

Hontarako, erabiltzaileak bi geruza zehaztu beharko ditu: sarrera geruza (ebakia izango dena) eta ebakitze geruza. Ebakitze geruza barnean dauden sarrera geruzako elementuak hautatuko dira. Elementuren baten zatiren bat ebakitze geruzaren kanpoaldean geratzen bada, hau, moztua izango da.

Irteera geruzak sarrerakoaren taula berdina izango du; desagertzen diren poligonoen datuak ez dira agertuko.

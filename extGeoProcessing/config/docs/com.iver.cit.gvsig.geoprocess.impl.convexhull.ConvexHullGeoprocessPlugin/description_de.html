Konvexe Hülle (Convex Hull)

Diese Funktion berechnet die konvexe Hülle um alle Geometrien der Eingabe-Ebene. Die Funktion „Konvexe Hülle“ benötigt als Eingabeparameter nur eine Vektorthema, welches sowohl vom Typ Punkt, Linie als auch Polygon sein kann. Optional kann auch die konvexe Hülle einer Auswahl der Eingabe-Ebene berechnet werden.

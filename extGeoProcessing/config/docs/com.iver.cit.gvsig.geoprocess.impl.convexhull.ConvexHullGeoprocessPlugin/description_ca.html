Embolcall convex

Este geoprocés calcula l'envoltant convex (convex buc), o polígon convex de menor àrea que embolica tots els elements vectorials d'una capa d'entrada.

Opera únicament amb una capa d'entrada, el tipus de geometria de la qual podrà ser de qualsevol tipus.

Les aplicacions d'este geoprocés poden ser de distint tipus:

  • Determinar la zona de cobertura d'un determinat fenomen geogràfic.

  • Càlcul del diàmetre de la zona coberta per una sèrie de geometries,

  • Etc.

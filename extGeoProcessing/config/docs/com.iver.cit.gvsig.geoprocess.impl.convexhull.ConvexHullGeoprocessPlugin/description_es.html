Envoltura Convexa

Este geoproceso cálcula la envolvente convexa (convex hull), o polígono convexo de menor área que envuelve a todos los elementos vectoriales de una capa de entrada.

Opera únicamente con una capa de entrada, cuyo tipo de geometría podrá ser de cualquier tipo.

Las aplicaciones de este geoproceso pueden ser de distinto tipo:

  • Determinar la zona de cobertura de un determinado fenómeno geográfico.

  • Cálculo del diametro de la zona cubierta por una serie de geometrías,

  • etc.

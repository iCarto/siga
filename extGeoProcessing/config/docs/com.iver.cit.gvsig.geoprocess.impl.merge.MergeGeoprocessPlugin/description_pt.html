JUNÇÃO / MERGE

Este geoprocesso atua sobre uma ou várias camadas, gerando uma nova camada que une todas as geometrias da camada de entrada.

A camada resultante deste geoprocesso conservará os atributos de uma das camadas de entrada, a especificada pelo usuário. O resto das camadas de entrada (as não selecionadas), conservarão apenas os atributos cujo nome e tipo de dado coincida com algum dos da camada de entrada selecionada pelo usuário.

    Este geoprocesso é útil, por exemplo, quando dispomos de uma série cartográfica, separada por folhas, e desejamos juntar o conteúdo das diferentes folhas em uma só camada.

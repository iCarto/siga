Sloučení

Tato prostorová překryvná operace pracuje s mnoha vstupními vrstvami. Všechny se sloučí do jedné výsledné vrstvy, která je sestavena z geoprvků všech vstupních vrstev. Protože každá vstupní vrstva může mít vlastní strukturu atributů, uživatel musí zvolit jednu vrstvu jejíž struktura atributů bude zachována a bude sloužit jako šablona. Atributy zbývajících vrstev budou zachovány pouze pokud se shodují názvem a typem se zvolenou šablonou.

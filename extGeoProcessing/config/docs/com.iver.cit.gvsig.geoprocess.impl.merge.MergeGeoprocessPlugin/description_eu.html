Elkartu

Geoprozesu honek bi geruza edo gehiagorekin egiten du lan. Irteera geruzan sarrerakoen elementu guztiak agertuko dira.

Irteera geruzako taula, erabiltzaileak zehaztutako sarrera geruza baten taulako datuez osatuko da.

Gainontzeko sarrera geruzen taula datuak, zehaztutakoaren izen eta era berekoak direnean baino ez dira agertuko irteera taulan.

Prozesu hau oso erabilgarri suertatzen da orri ezberdinetan banatutako kartografia elkartzerakoan.

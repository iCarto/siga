Este geoprocés actua sobre una o diverses capes, generant una nova capa que unisca totes les geometries de la capa d'entrada. La capa resultant d'este geoprocés conservarà els atributs d'una de les capes d'entrada, l'especificada per l'usuari. De la resta de capes no seleccionades, es conservaran aquells atributs el nom dels quals i els tipus de dada coincidisca amb algun dels de la capa seleccionada per l'usuari.

Este geoprocés és d'utilitat, per exemple, quan ens arriba una sèrie cartogràfica, separada per fulls, i desitgem ajuntar el contingut dels diferents fulls en una sola capa. Este és el cas de la sèrie de fulls Magnes, publicada pel ITGME espanyol (Institut Tecnològic i Geominer d'Espanya).

Złączenie

Geoproces ten może działać z wieloma warstwami wejściowymi. Złącza on je wszystkie tworząc warstwę wynikową zawierającą obiekty wszystkich warstw wejściowych. Ponieważ każda z nich może mieć różne atrybuty użytkownik musi wybrać warstwę, której schemat atrybutów zostanie nadany warstwie wynikowej. Pozostałe warstwy jeśli tylko będą zawierać atrybuty takie same jak wybrany schemat, jej wartości zostaną przpisane do warstwy wynikowej.

Themen zusammenführen (Merge)

Die Funktion „Themen zusammenführen“ kombiniert zwei oder mehrere Eingabe-Ebenen. Das neue Vektorthema enthält dann ein Element für jedes Element der Eingabe-Ebenen. Da nur ein Tabellenschema übernommen werden kann, muss der Benutzer ein zu übernehmendes Schema auswählen. Attribute weiterer Eingabe-Ebenen werden nur dann übernommen, sofern diese über denselben Namen und Typ des ausgewählten Tabellenschemas verfügen.

Průnik

Tato prostorová překryvná operace pracuje se dvěmi vrstvami: vstupní a překryvná vrstva.

Pro každý geoprvek vstupní vrstvy se vypočítá průsečík s geoprvky překryvné vrstvy. Pro každý průnik je vytvořen nový geoprvek, který přebírá strukturu atributů z obou vstupních vrstev. Nazývá se to “Prostorová AND”, protože všechny geoprvky výsledné vrstvy jsou společné oběma vrstvám.

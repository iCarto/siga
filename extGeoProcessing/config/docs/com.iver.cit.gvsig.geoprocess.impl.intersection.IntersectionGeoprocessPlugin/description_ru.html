Пересечение (Intersect)

Этот инструмент работает с двумя слоями - исходным слоем и операционным слоем.

Операция соответствует запросу "Spatial AND". Для каждого объекта исходного слоя рассчитывается пересечение с объектами операционного слоя. Для каждого пересечения создается новый объект с рассчитанным пересечением и атрибутами исходного объекта.

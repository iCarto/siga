INTERSECCIÓ

Este geoprocés opera sobre dos capes, la capa d'entrada i la capa de cavalcament, les geometries de la qual han de ser forçosament poligonals.

Per a cada geometria de la capa d'entrada, calcula la intersecció amb les diferents geometries de la capa de cavalcament, i origina un nou element per cada intersecció. Este element prendrà tots els atributs alfanumèrics de les geometries que el van originar (d'entrada i cavalcament). Per este motiu (modela zones de l'espai que complixen la condició de pertànyer als dos polígons que l’han originat) a este geoprocés se’l coneix com a operador AND espacial.

Un exemple d'aplicació d'este geoprocés seria per a obtindre, donada una capa d'usos del sòl Corine 2000, i una capa del mapa geològic nacional, una capa de polígons amb informació homogènia d'ús del sòl i material geològic.

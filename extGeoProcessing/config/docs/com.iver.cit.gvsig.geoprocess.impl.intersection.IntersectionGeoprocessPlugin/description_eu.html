Elkargunea

Elkargune geoprozesuak bi geruzekin egiten du lan: sarrera geruza eta gainjartze geruza. Bai sarrera geruza eta baita gainjartze geruza ere poligonodunak izan behar dira.

Elkargune geoprozesua ‘AND’ espaziala bezala ezagutzen da: Sarrera geruzan ETA gainjartze geruzan dauden guneak aukeratzen ditu. Irteera geruzako taulan sarrera geruza eta gainjartze geruzako datuak agertuko dira.

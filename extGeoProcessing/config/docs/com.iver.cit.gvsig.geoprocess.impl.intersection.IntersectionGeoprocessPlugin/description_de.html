Verschneidung (Intersect)

Diese Funktion verschneidet zwei Vektorthemen A („Eingabe-Ebene“) und B („Overlay-Ebene“) so, dass für jede Verschneidung ein neues Element erzeugt wird. Das Ergebnisthema enthält nur Flächen, die sowohl in A als auch in B enthalten sind. Bei dieser Funktion werden die Attribute beider Augangsthemen im Ergebnis übernommen.

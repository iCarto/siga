Przestrzenne złączenie

Geoproces ten przenosi atrybuty z warstwy źródłowej do docelowej (podobnie jak w złączeniu atrybutowym). Różnica między tradycyjnym złączeniem polega na kryteriach złączenia, którymi nie są wartości atrybutów, lecz kryteria przestrzenne: przecięcie lub sąsiedztwo.

Jeśli użytkownik wybierze przecięcie jako kryterium złączenia, każdy obiekt warstwy docelowej może przecinać wiele obiektów warstwy źródłowej (relacja 1-N). Jeśli użytkownik wybierze sąsiedztwo powstaje relacja 1-1.

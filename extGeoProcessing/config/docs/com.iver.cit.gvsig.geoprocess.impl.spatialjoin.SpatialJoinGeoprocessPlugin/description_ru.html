Пространственное объединение

Этот инструмент присваивает атрибуты входному набору данных из операционного набора данных на основе их пространственного расположения относительно друг друга.

Каждому объекту входного набора данных будут присвоены суммарные числовые атрибуты объекта операционного слоя.
Дополнительно можно указать "Использовать ближайшую геометрию". Тогда каждому объекту входного набора данных будут присвоены атрибуты ближайшего объекта операционного слоя, с указанием в отдельном поле расстояния до него.

Bufor

Geoproces te tworzy nowa warstwę poligonów zawierającą żądane bufory wybranych obiektów warstwy wejściowej.

Obiekty warstwy wejściowej mogą być punktowe, liniowe lub powierzchniowe. Można utworzyć jeden lub wiele pierścieni buforu dla obiektów. Dodatkowo jeśli warstwa wejściowa jest zawiera poligony, bufor może być wewnętrzny lub zewnętrzny (lub oba naraz).

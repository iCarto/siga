Obalová zóna

Tento geoproces vytvoří novou polygonovou vrstvu s polygonovými buffery kolem prvků vstupní vrstvy.

Prvky vstupní vrstvy mohou být různého typu (bod, linie nebo polygon). Pro každý vstupní prvek můžete vytvořit jednu nebo více ekvidistantních obalových zón. Pokud tedy vstupní prvek je typu polygon, obalová zóna může být vnitřní, vnější nebo obojí.

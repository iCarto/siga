Unió

Este geoprocés, igual que els geoprocessos Intersecció i Diferència opera sobre dos capes de polígons, i s’obtenen les seues interseccions (per este motiu, a estos tres geoprocessos se'ls coneix com a "geoprocessos de cavalcament").

Al geoprocés Unió se’l coneix com a "OR espacial", perquè la capa de resultat estarà formada per les geometries que apareixen en les dos capes (interseccions entre els polígons), més les geometries que apareixen només en una o altra de les dos capes posades en relació. Açò es traduïx que el geoprocés realitza tres passades: la primera per a calcular la intersecció d'ambdós capes, la segona per a calcular les diferències de la primera amb la segona i la tercera passada per a calcular les diferències de la segona capa amb la primera.

Este geoprocés sempre que ens interesse generar noves capes que posen de manifest la idea de dos fenòmens, de manera que es ressalte la idea d'algun dels dos (o dels dos).

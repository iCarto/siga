Spojení

Podobně jako průnik a rozdíl se jedná o prostorovou překryvnou operaci. Pracuje se vstupní a překryvnou vrstvou.

Nazývá se “Prostorové OR” protože geoprvky výstupní vrstvy jsou brány z kombinace vstupní a překryvné vrstvy. Struktura atributů a jejich hodnoty se přenáší pouze ze vstupní vrstvy.

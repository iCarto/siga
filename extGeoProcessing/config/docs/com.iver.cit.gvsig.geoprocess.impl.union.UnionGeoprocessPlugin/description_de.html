Vereinigung (Union)

Die Funktion „Vereinigung“ benötigt als Eingabeparameter zwei Vektorebenen - eine Eingabe-Ebene A sowie eine Overlay-Ebene B. Beide Vektorebenen werden derart miteinander verschnitten, so dass das Ergebnis Elemente enthält, die aus einem der beiden Themen A oder B stammen. Die Funktion „Vereinigung“ wird daher auch als „Spatial OR“ bezeichnet.

Reprojecció

Este geoprocés permet canviar la projecció geodèsica dels elements vectorials de la capa d'entrada. Per a este fet, l'usuari haurà d'especificar la nova projecció que s’ha d’aplicar.


Este geoprocés pot ser de gran utilitat per a fer concordar cartografies en un mateix projecte cartografies que es troben en projeccions distintes.

REPROJEÇÃO / REPROJECTION



Este geoprocesso permite alterar o sistema de projeção cartográfica dos elementos vetoriais da camada de entrada. Para isso, o usuário deverá especificar a nova projeção que deseja aplicar.

    Este geoprocesso pode ser de grande utilidade para que haja concordância na sobreposição de camadas que se encontram em projeções distintas em um mesmo projeto de gvSIG.

Traslación 2D

Este geoproceso permite aplicar una transformación de traslación sobre todos los puntos de las geometrías de la capa de entrada. Para tal fin, el usuario deberá especificar el desplazamiento en X y en Y a aplicar.

Este geoproceso puede ser de gran utilidad para hacer concordar cartografías procedentes de fuentes distintas, en lo que se viene a denominar por el término inglés conflation.

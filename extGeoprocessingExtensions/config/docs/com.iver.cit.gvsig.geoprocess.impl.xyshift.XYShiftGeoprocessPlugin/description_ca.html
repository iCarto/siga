Translació 2D

Este geoprocés permet aplicar una transformació de translació sobre tots els punts de les geometries de la capa d'entrada. Per a este fi, l'usuari haurà d'especificar el desplaçament en X i en I que s’ha d’aplicar.

Este geoprocés pot ser de gran utilitat per a fer concordar cartografies procedents de fonts distintes, que es denomina amb el terme anglés conflation.

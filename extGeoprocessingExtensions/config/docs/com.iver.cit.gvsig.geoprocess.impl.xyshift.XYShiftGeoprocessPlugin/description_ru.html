XY смещение

Эта операция применяет 2D (двумерное, плоскостное) преобразование (трансформацию) ко всем объектам исходного слоя. Для выполнения задачи, пользователь должен ввести смещения по осям X и Y.

Это полезно при интеграции (объединении) слоев из разных источников и с разными картографическими характеристиками.

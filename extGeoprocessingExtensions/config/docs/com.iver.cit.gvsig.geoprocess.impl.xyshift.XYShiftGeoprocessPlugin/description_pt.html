TRANSLAÇÃO 2D / TRANSLATION 2D

Este geoprocesso permite aplicar uma transformação de translação 2D sobre todos os pontos das geometrias da camada de entrada. Para isso, o usuário deverá especificar o deslocamento nos eixos X e Y que deseja aplicar.



Este geoprocesso pode ser de grande utilidade para que haja concordância na sobreposição de camadas procedentes de fontes distintas, denominado pelo termo em inglês conflation.

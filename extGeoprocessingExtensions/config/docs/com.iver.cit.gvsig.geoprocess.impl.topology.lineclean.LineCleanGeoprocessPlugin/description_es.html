Clean

Este geoproceso corrige topológicamente las geometrías de una capa de líneas, de forma que esta capa pase a tener una topología de líneas correcta.

Por "topología correcta de líneas", entendemos que las geometrías lineales de esta capa no se autointersecten, y que solamente intersecten con otras geometrías de líneas por sus puntos extremos (nodos).

Si durante la ejecución del mismo se detecta que dos líneas intersectan en un punto que no es nodo (pseudonodo), se fragmentará estas geometrías originando cuatro nuevas líneas que sí intersectarán en sus extremos.

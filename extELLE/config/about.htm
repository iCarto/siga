ELLE

Description:

ELLE (Easy Load Layer Extension) allows custom maps loading and saving from a PostGIS database.

Features:

  • User-defined map loading.
  • Styles (gvl, sdl) storage in data base and filesystem.
  • Load layers from any stored map, without loading all map.
